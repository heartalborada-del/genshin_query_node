# 这是什么
这是一个可以提供网页数据查询的小工具/提供查询api
# 如何使用
自己搭建或使用我提供的网页 [戳我直达](http://genshin-query-node.vercel.app/)
# 如何搭建
- 搭建api
  - CN(国服)
    1. 将`index.js`里面的`openCN`设置为true
    2. `node index.js`即可
  - OS(国际服)
    1. 将`index.js`里面的`openOS`设置为true
    2. `node index.js`即可
- 使用网页查询
    1. 将`index.js`里面的`openWEB`设置为true
    2. `node index.js`即可
